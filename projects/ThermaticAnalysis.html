Thematic Analysis Web Application


1. Conduct Thematic Analysis in Excel

Begin by identifying themes, factors, and raw codes within Excel. Use structured sheets to categorize data extracted from source material.


2. Update Factors and Related Groups/Sub-Groups

Revise or add new factors in the spreadsheet and associate them with the correct group and subgroup structures for deeper thematic insight.


3. Modify Group/Sub-Group Definitions

Edit or refine the naming and grouping structures to better reflect emerging themes during analysis.


4. Optionally Add Categories and Subcategories

Add higher-level categories and subcategories if your analysis benefits from hierarchical organization beyond groups.


5. Add Search Metadata for Categories/Subcategories

Enhance your data structure by assigning searchable tags to factors. These will assist in filtering and future retrieval within the web app.


6. Export Data to Webpage

Use the provided Python scripts to extract, clean, and convert Excel data into JSON. Then populate the HTML structure dynamically with this content.


7. Review Glossary and Web Analysis

The final HTML output includes a glossary of terms and grouped factors, allowing for user-friendly browsing and review of the thematic structure.
